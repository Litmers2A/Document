"작가도 메뉴중에 마이페이지를 추가해서, 마이페이지로 연결되도록 해야해. 작가가 마이페이지에서 해야하는 것:
1. 계정 설정 및 환경 설정 2. 작가소개글 쓰기"

## 영향 범위 분석 (Impact Analysis)

### 화면 (Routes)
| Route | 변경 내용 | 신규 여부 |
|-------|----------|---------|
| /mypage | 작가 Role 접근 가능하도록 확장 (현재 독자 전용) | No (확장) |
| /mypage/settings | 작가소개글 편집 섹션 추가 | No (확장) |

### 컴포넌트 (Components)
| Component | 변경 내용 | 신규 여부 |
|-----------|----------|---------|
| Creator Layout (`app/(creator)/layout.tsx`) | studioNavItems에 "마이페이지" 항목 추가 | No |
| MyPage (`app/(auth)/mypage/page.tsx`) | 작가 Role 전용 콘텐츠 표시 (조건부 렌더링) | No |
| Settings (`app/(auth)/mypage/settings/page.tsx`) | 작가소개글 편집 섹션 추가 | No |
| CreatorBioSection | 작가소개글 편집 UI (Textarea, Plain Text, 500~1000자) | Yes |

### 상태/Interaction
| Hook/Store | 변경 내용 |
|------------|----------|
| useCreatorBioUpdate | 작가소개글 업데이트 API 호출 및 authStore 동기화 (신규) |
| authStore | User 타입에 작가소개글 필드 추가 (creatorBio?: string) |

### 스타일/UX
| 항목 | 변경 내용 |
|------|----------|
| Creator SideNav | "마이페이지" 메뉴 항목 추가 (아이콘: User 또는 UserCircle) |
| MyPage 레이아웃 | Role별 조건부 콘텐츠 (독자: 지갑 바로가기 / 작가: 작가소개글 카드) |
| Settings 페이지 | 작가소개글 편집 섹션 추가 |
| 작가소개글 에디터 | Textarea UI (Plain Text, 500~1000자 제한, 실시간 글자수 카운터) |

## 모듈 의존성

| From | To | 이유 |
|------|-----|------|
| Creator Layout | /mypage | 작가 메뉴에서 마이페이지로 링크 |
| MyPage | authStore | 현재 사용자의 Role 확인 (독자 vs 작가) |
| Settings | CreatorBioSection | 작가소개글 편집 섹션 통합 |
| CreatorBioSection | useCreatorBioUpdate | 소개글 업데이트 로직 |
| useCreatorBioUpdate | profileService (확장) | 작가소개글 업데이트 API 호출 |
