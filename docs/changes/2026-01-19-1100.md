## 원문 (Raw Feedback)

### feedback-250119-1.md: Proposal · Backing 정산 및 알림 정책

**핵심 원칙:**
1. deal / drop 선택 시점마다 독자에게 즉시 알림 발송
2. Ink 입·출금 확정은 오직 작품이 `published`로 전환되는 순간에만 발생
3. 작품이 `published`로 전환될 때는, Proposal 상태와 무관하게 '금전 결과 확정 알림'을 반드시 발송
4. `published + null` 상태는 drop과 동일하게 처리 및 알림

**상태 정의:**
- Proposal Status: `deal` (수락), `drop` (거절), `null` (미결정)
- Work Status: `open` (공개 전), `published` (공개 완료)

### feedback-250119-2.md: 제안 내역 및 후원 상세 페이지 구현

**진입 경로:**
1. 마이페이지: "제안 내역" 버튼 추가
2. 내 지갑: BalanceCard의 "진행 중(Pending Ink)" 버튼 클릭 시 `/mypage/proposals?filter=pending`으로 연결

**페이지 구성:**
- 제안 내역 목록 페이지: 필터(전체/진행 중/완료), 카드 항목(작품 제목, Ink 금액, 상태 태그)
- 제안 상세 페이지: 상단 요약 영역(결과 스테이지 3단계), 타임라인(Status History)

**상태별 문구 매핑:**
| Proposal Status | Work Status | 표시 상태 | 안내 메시지 |
|-----------------|-------------|----------|-----------|
| null | open | 검토 중 | 작가가 제안을 확인하고 있습니다. |
| deal | open | 성사 대기 | 작가가 제안을 수락했습니다. 작품이 공개되면 정산됩니다. |
| drop | open | 거절됨 (반환 대기) | 작가가 제안을 거절했습니다. 작품이 공개되면 Ink가 반환됩니다. |
| deal | published | 후원 성공 | 작품이 공개되어 Ink 정산이 완료되었습니다. |
| drop/null | published | 환불 완료 | 제안이 채택되지 않아 Ink가 전액 반환되었습니다. |
