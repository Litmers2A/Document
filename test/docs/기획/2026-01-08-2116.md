## 원문 (Raw Feedback)

```
모바일 화면일 때는 하단네비게이션바가 나와야하는데, 데스크탑 화면일 때는 하단네비게이션바가 사라지고, 왼쪽에 메뉴사이드바가 들어가야해.
화면 크기와 관계없이 상단바 들어가야해. 상단바에는 홈화면으로 리다이렉트 시켜주는 로고와 검색바가 있어야해
```

### 컴포넌트 (Components)

| Component | 변경 내용 | 신규 여부 |
|-----------|----------|---------|
| BottomNav.tsx | 데스크탑에서 숨김 처리 (`lg:hidden` 추가) | No |

| SideNav.tsx | 독자 영역에서도 사용 (navItems 정의 필요) | No (재사용) |

| Hook/Store | 변경 내용 |

- (선택) 실시간 자동완성 (TBD)

### 스타일/UX

| 항목 | 변경 내용 |
|------|----------|
| 상단바 높이 | 전역 h-14 또는 h-16 고정, main 영역 padding 조정 불필요 (sticky) |
| 하단 네비게이션 | 모바일: 표시, 데스크탑: 숨김 (`lg:hidden`) |
| 사이드바 | 모바일: 숨김, 데스크탑: 표시 (`hidden lg:flex`), 왼쪽 고정 (w-64) |
| 메인 콘텐츠 영역 | 데스크탑에서 사이드바 너비만큼 왼쪽 패딩 (`lg:pl-64`) |
| 반응형 브레이크포인트 | `lg` (1024px) 기준으로 모바일/데스크탑 분기 |

**레이아웃 구조 (변경 후)**:

```
모바일 (< 1024px):
┌─────────────────┐
│   TopNav (상단바)  │ ← 로고 + 검색바
├─────────────────┤
│                 │
│   Main Content  │
│                 │
├─────────────────┤
│  BottomNav (하단)│ ← Home/Explore/MyPage
└─────────────────┘

데스크탑 (≥ 1024px):
┌───────┬─────────────────┐
│       │   TopNav (상단바)  │ ← 로고 + 검색바
│       ├─────────────────┤
│ Side  │                 │
│ Nav   │   Main Content  │
│ (왼쪽) │                 │
│       │                 │
└───────┴─────────────────┘
```
