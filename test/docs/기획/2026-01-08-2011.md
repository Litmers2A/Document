# Change Analysis: Typography Hierarchy Enhancement for Creator Studio & Admin

> 생성일시: 2026-01-08 20:11
> 상태: applied

## 원문 (Raw Feedback)

크리에이터 스튜디오 및 어드민 화면 전반에서
텍스트 크기와 무관하게 font-weight가 거의 동일하게 사용되고 있어,
텍스트 위계가 시각적으로 충분히 드러나지 않는 문제가 있습니다.

특히 **페이지 제목, 섹션 헤더, 테이블 헤더**처럼
사용자가 빠르게 스캔하며 구조를 파악해야 하는 요소들은
본문 텍스트보다 **명확히 더 무거운 font-weight**를 가져야 합니다.

현재 구조에서는

* 제목과 본문이 한 덩어리로 읽히고
* 화면 진입 시 "어디가 제목인지" 즉시 구분되지 않아
  정보 파악에 불필요한 인지 비용이 발생합니다.

👉 제목 / 헤더 / 본문 텍스트 간에
**font-size + font-weight 조합으로 명확한 타이포그래피 위계**를 정의해 주세요.
(예: Heading은 동일 size라도 더 높은 weight를 사용)
