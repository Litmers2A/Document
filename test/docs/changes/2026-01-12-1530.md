# Change Analysis: 신고 기능 추가

> 생성일시: 2026-01-12 15:30
> 상태: applied

## 원문 (Raw Feedback)

유저가 신고할 수 있는 것: 스토리, proposalcard의 내용을 쓴 proposal한 사용자
작가 또한 proposalcard의 내용을 보고, proposal한 사용자를 신고할 수 있어
관리자가 <신고 관리> 탭에서 확인해야 하는 것 : 신고한 유저, 신고를 받은 내용 및 이유

## 확정 사항 (User 결정)

| 항목 | 결정 내용 |
|------|----------|
| 스토리 신고 대상 | 스토리 콘텐츠 자체 |
| 신고 유형 | 6가지 (스팸, 욕설/비방, 부적절한 콘텐츠, 저작권 침해, 허위 정보, 기타) |
| 작가 신고 범위 | 본인 스토리에 달린 Proposal만 |
| 처리 알림 | 미발송 (마이페이지에서 확인) |
