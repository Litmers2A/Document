## 페이지별 와이어프레임 설명

### 1. 메인 홈 (`/`)

#### 섹션 구성

**[SECTION 01] Hero Banner**
- 전체 화면 비디오 또는 고화질 이미지 슬라이더
- 슬라이드 수: 3개
- 텍스트 오버레이: 브랜드 슬로건 + CTA 버튼 ("쇼핑하기", "브랜드 스토리")
- 자동 전환: 5초


**[SECTION 02] 베스트셀러**
- 제품 카드 4개 가로 나열
- 각 카드: 제품 이미지 / 제품명 / 가격 / 찜 아이콘 / 장바구니 추가 버튼
- "전체 보기" 링크

```
[더미 제품 데이터]
1. 글로우 리뉴얼 세럼 30ml — 89,000원 ⭐ 4.9 (리뷰 312개)
2. 딥 하이드레이션 크림 50ml — 76,000원 ⭐ 4.8 (리뷰 241개)
3. 소프트 밸런싱 토너 150ml — 52,000원 ⭐ 4.7 (리뷰 198개)
4. 선 프로텍터 SPF50+ PA++++ 50ml — 45,000원 ⭐ 4.8 (리뷰 403개)
```

**[SECTION 03] 브랜드 철학 인트로**
- 배경: 크림 화이트
- 텍스트 중앙 정렬, 세리프 폰트
- 짧은 카피 + 브랜드 스토리 링크

**[SECTION 04] 성분 하이라이트**
- 3열 아이콘 카드
- 주요 성분: 세네데름 발효 추출물 / 히알루론산 3중 복합체 / 폴리글루타믹산

**[SECTION 05] 매거진 프리뷰**
- 최신 아티클 3개 썸네일 카드
- 카테고리 태그 + 제목 + 날짜

**[SECTION 06] 인스타그램 피드 연동**
- 최근 9개 게시물 3×3 그리드
- 클릭 시 인스타그램 이동

**[SECTION 07] 뉴스레터 구독**
- 이메일 입력 폼
- 문구: "가장 먼저 신제품과 스킨케어 팁을 받아보세요."

### 3. 제품 상세 (`/products/skincare/:id`)

#### 레이아웃 구성

```
[좌측 — 제품 이미지]
- 메인 이미지 (정사각형, 1:1)
- 썸네일 이미지 4–6개 하단 나열
- 360° 뷰 또는 확대 기능

[우측 — 제품 정보]
- 브랜드명 (소문자, 라이트 웨이트)
- 제품명 (대형 타이틀)
- 별점 + 리뷰 수 (앵커 링크)
- 가격
- 용량 선택 (30ml / 50ml / 100ml)
- 수량 선택
- [장바구니 담기] 버튼 (Primary)
- [바로 구매] 버튼 (Secondary)
- 찜하기 아이콘

[하단 탭 영역]
- 탭1: 제품 설명
- 탭2: 성분 전성분표
- 탭3: 사용 방법
- 탭4: 리뷰 (별점 분포 + 리뷰 목록)
- 탭5: 문의
```

#### 더미 제품 상세 — 글로우 리뉴얼 세럼

```markdown
**제품명**: 글로우 리뉴얼 세럼
**부제**: 발효 바이오틱 복합체로 빛나는 피부 결

**제품 설명**:
피부 고유의 재생 메커니즘에 착안한 글로우 리뉴얼 세럼은
LUMIÈRE 특허 성분 '세네데름 발효 추출물'을 고농도로
담아냈습니다. 얇은 막 질감으로 빠르게 흡수되어 촉촉하고
윤기 있는 피부 결을 하루 종일 유지시켜 줍니다.

**주요 성분**:
- 세네데름 발효 추출물 12% (특허 성분)
- 히알루론산 3중 복합체
- 나이아신아마이드 5%
- 판테놀

**사용 방법**:
토너 사용 후 적당량(2–3방울)을 손바닥에 덜어
얼굴 전체에 고르게 펴 바른 후 가볍게 두드려 흡수시킵니다.
아침·저녁 사용을 권장합니다.

**전성분**: 정제수, 부틸렌글라이콜, 글리세린, 세네데름발효여과물,
나이아신아마이드, 히알루론산나트륨, 소듐PCA, 판테놀,
알란토인, 카보머, 아르기닌, 향료, 페녹시에탄올

**용량 / 가격**:
- 30ml / 89,000원
- 50ml / 138,000원
```

## 주요 UX 플로우

### 구매 플로우

```
홈 또는 제품 목록
  → 제품 상세 페이지
    → 옵션 선택 (용량, 수량)
      → [장바구니 담기] or [바로 구매]
        → 로그인 체크 (비회원 구매 가능)
          → 배송지 입력
            → 결제 수단 선택
              → 주문 완료 페이지 (이메일 자동 발송)
```

### 회원가입 플로우

```
회원가입 버튼 클릭
  → 소셜 가입 (카카오 / 네이버 / 구글) or 이메일 가입
    → 이메일 인증
      → 기본 정보 입력 (이름, 생년월일, 피부타입 선택)
        → 마케팅 수신 동의
          → 가입 완료 (웰컴 쿠폰 발급)
```
