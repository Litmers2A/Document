## TBD (미결정 사항) → ✅ 결정 완료

### ✅ TBD-001: 프로필 이미지 파일 크기/형식 제한 정책
**결정**:
- 허용 형식: JPG, JPEG, PNG, WEBP
- 최대 용량: 3MB
- 권장 비율: 정사각형 (1:1)
- 권장 해상도: 256×256 ~ 512×512
- 기본값: 랜덤 기본 아바타 제공 (이미지 없이 활동 가능)

**이유**: 커뮤니티 특성상 프로필 이미지 중요도 낮음. 익명성 보호 + 업로드 허들 최소화.

### ✅ TBD-002: 닉네임 유효성 검사 규칙
**결정**:
- 길이: 2~12자
- 허용 문자: 한글, 영문, 숫자, `_` `.` 만 허용
- 불가: 공백, 이모지, 연속 특수문자 (`__`, `..`)
- 중복 허용 여부: ❌ (닉네임 고유)
- 금지어: 욕설, 성적 표현, 작가/작품 사칭

**이유**: 웹소설/웹툰 커뮤니티는 닉네임 = 정체성. 12자 제한 → 댓글/게시판 UI 안정.

### ✅ TBD-003: 약관 동의 항목의 필수/선택 여부
**결정**:
- **필수**: 서비스 이용약관, 개인정보 처리방침, 커뮤니티 이용 규칙
- **선택**: 마케팅 정보 수신 동의 (이메일/푸시)

**이유**: 커뮤니티는 운영 규칙 동의가 매우 중요. 분쟁·제재 시 근거 확보용.

### ✅ TBD-004: 약관 전문 페이지 링크 필요 여부
**결정**: 필요 (O)
- 회원가입 화면: 요약 + 링크
- 전문 페이지: 별도 URL 제공
- 커뮤니티 이용 규칙은 게시판 상단에서도 재노출

**이유**: 분쟁 잦은 커뮤니티 특성. "몰랐다" 방어 차단용.

### ✅ TBD-005: Settings 페이지 접근 경로
**결정**:
- **Primary**: MyPage → Settings
- **Secondary**: TopNav 프로필 드롭다운 → Settings
- **모바일**: 햄버거 메뉴 → MyPage → Settings

**이유**: 커뮤니티 메인 UX는 콘텐츠 중심. TopNav에 Settings 단독 노출 ❌ (몰입 방해).

### ✅ TBD-006: 프로필 수정 시 비밀번호 재확인 여부
**결정**:
- **기본 정보** (닉네임, 프로필 이미지) → ❌ 재확인 없음
- **보안 민감 정보** (이메일 변경, 비밀번호 변경, 탈퇴) → ✅ 비밀번호 재확인 필수

**이유**: 커뮤니티는 수정 빈도 높음. 모든 수정에 재인증 → UX 급락. 보안/UX 균형점.
