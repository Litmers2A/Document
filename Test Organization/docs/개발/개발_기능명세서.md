## 기능 상세

### F-01 · GNB 네비게이션

- 스크롤 시 배경 투명 → 크림 화이트로 전환 (transition 0.3s)
- 모바일: 햄버거 메뉴 → 풀스크린 오버레이 드로어
- 장바구니 아이콘에 수량 뱃지 표시

### F-02 · Hero 배너

- 슬라이드 3개, 자동 전환 5초, 페이드 트랜지션
- 영상 소재: MP4 autoplay / muted / loop
- 모바일에서는 이미지 폴백 처리

### F-03 · 제품 목록 필터 & 정렬

- URL 쿼리스트링 기반 (`?category=serum&skin=dry&sort=review`)
- 필터 선택 시 페이지 새로고침 없이 결과 업데이트 (클라이언트 사이드 필터링)
- 선택된 필터 태그 상단 표시 + 개별 제거 가능

### F-04 · 제품 상세 탭 UI

- 탭: 제품 설명 / 전성분표(기본 펼침) / 사용 방법 / 리뷰 / 문의
- URL 해시 기반 탭 상태 유지 (`#ingredients`)
- 리뷰 탭: 별점 분포 바 차트 + 페이지네이션 (10개씩)

### F-05 · 모바일 하단 고정 CTA 바

- 제품 상세 페이지 모바일(~768px)에서만 노출
- 버튼 2개: [장바구니 담기] [바로 구매]
- 스크롤 최상단에서는 숨김 처리

### F-08 · 결제 (토스페이먼츠 연동)

- 토스페이먼츠 SDK v2 사용
- 지원 결제 수단: 카드 / 카카오페이 / 네이버페이 / 무통장입금
- 결제 완료 시 주문 확인 이메일 자동 발송 (SendGrid)
