# Change Analysis: Ink 충전 카드 UI 개선

> 생성일시: 2026-01-08 20:54
> 상태: applied

## 원문 (Raw Feedback)

ink 충전 페이지에서,
각 선택지 카드 버튼들은 모두 **동일한 컴포넌트**로 사용되고 있는데,
현재는 **카드 내부 콘텐츠 길이에 따라 높이가 달라져**
그리드 상에서 시각적 정렬이 깨지고 있습니다.

예를 들어,
`1,000 Ink` 카드와 `5,500 Ink` 카드의 **전체 높이가 서로 달라**
나란히 배치되었을 때 하나의 선택 옵션 묶음으로 인식되기 어렵습니다.

👉 동일한 컴포넌트인 만큼
**카드 전체 높이를 고정하거나**,
**내부 요소 간 spacing을 조정해 모든 카드가 동일한 높이**를 가지도록 맞추는 것이 좋겠습니다.
(보너스 배지 유무와 관계없이)

또한 카드 하단의 **가장 작은 설명 텍스트(기본 Ink / 보너스 Ink 등)**는
현재 크기가 너무 작아 가독성이 떨어지고,
정보가 '보조 정보'가 아니라 '무시 가능한 정보'처럼 인식됩니다.

👉 보조 텍스트라도
* 최소 가독 가능한 font-size를 유지하고
* 카드 간 비교 시 의미를 전달할 수 있도록
  **텍스트 크기 또는 line-height를 소폭 상향**해 주세요.
